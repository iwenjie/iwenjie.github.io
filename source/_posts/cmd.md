cmd窗口命令
<!--more-->

```
calc // 打开计算器
notepad // 打开记事本
ipconfig  // 查看IP地址
netsh wlan show hostednetwork // 查看虚拟WiFi
netsh wlan start hostednetwork // 启动虚拟WiFi
netsh wlan stop hostednetwork // 停止虚拟WiFi
netsh wlan set hostednetwork mode=allow ssid=aaa password=dddd // 设置虚拟WiFi

cd /mulu // 进入目录
cd ../  // 返回上一级
tree // 查看文件夹树形结构
dir  // 查看目录
mkdir aaa // 创建一个aaa文件夹

```

#### hexo

hexo g // 编译
hexo s // 启动server
